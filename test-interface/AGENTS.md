## Project Context
**test-interface** is a lightweight playground for testing frontend components and API integrations. It uses Vanilla JS, HTML, and CSS.

## Coding Standards
- Keep it simple and dependency-free.
- Clean DOM manipulation.

## Jules Instructions
When reviewing:
1.  **Simplicity**: Avoid over-engineering.
2.  **Bug Detection**: Look for common JS errors in `app.js`.
3.  **Style**: Ensure `styles.css` is clean and follows basic UI principles.
