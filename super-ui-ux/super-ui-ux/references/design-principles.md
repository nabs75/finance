# Principes de Design UX

1. **Hiérarchie Visuelle** : Utilisez la taille et la couleur pour guider l'œil vers l'action principale.
2. **Loi de Hick** : Plus il y a de choix, plus le temps de décision augmente. Simplifiez.
3. **Contrastes** : Assurez un ratio minimal de 4.5:1 pour le texte normal.
4. **Consistance** : Un bouton doit ressembler à un bouton sur toutes les pages.
